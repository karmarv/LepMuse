# Datasets

This folder owns data and dataset-specific utilities.

- `battus10/`: small Battus training/validation set.
- `battus100/`: larger Battus training/validation set and manual measurement labels.
- `uf_museum_battus/`: UF Museum metadata, manifests, sample images, unseen-run edge-case outputs, and metadata preparation utilities.

UNet model files live under `segmentation/unet/models/`.
